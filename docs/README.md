## 🌟 系统简介

Star 是一个**专业级占星计算与可视化系统**，采用现代化的前后端分离架构，提供：

- **精确的天文计算**：基于 JPL 星历数据的行星位置计算
- **完整的占星功能**：本命盘、行运、推运、年限法
- **多时间粒度分析**：小时/日/周/月/年趋势分析
- **可视化界面**：专业的星盘图、时间线、趋势图
- **智能体集成**：支持 LLM 智能体接入

## 🚀 快速开始

### 后端 (Go)
```bash
cd Star.go
go build -o bin/star-api .
./bin/star-api
# 服务运行在 http://localhost:8080
```

### 前端 (Next.js)
```bash
cd Star
npm install
npm run dev
# 服务运行在 http://localhost:3000
```

## 📊 核心功能

| 功能模块 | 描述 |
|---------|------|
| 本命盘计算 | 行星位置、宫位、相位、图形相位 |
