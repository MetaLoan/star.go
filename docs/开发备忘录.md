关于分数计算：


1、得分的最小颗粒度是小时
2、分数计算来自于基础分+影响因子
3、影响因子由天文系统运转决定，因此不同影响因子有作用时长，某些影响因子变化比较快所以影响的比较快（所以会存在影响每个小时的数值变化），某些影响因子是按照年或者数十年计算的。
4、影响因子的影响程度不是一成不变的。他遵循影响开始-影响加强-影响减弱-影响小时的函数变化，如何设计这个函数？需要你进行合理设计并告知
5、某一时刻的分值是由各种影响因子叠加的结果并且在变化（因为影响因子在不断变化）
6、24小时的小时分决定当日分，7日分决定周分，30日分决定月分，365日分决定年分。数值的表现是互相关联的。就好比股票市场的5分钟K线，1小时日线，年K线。他们只是数值的周期性趋势展示，并没有重新计算。
7、根据第6点，因此趋势线图支持按小时/天/周/月/年进行分级查看，同时能够查看每一级别瞬时收到的影响因子有哪些，但是由于年线/周线/月线/日线/小时线受影响的因子比较多，为了便于展示和计算，影响因子你可以做如下分类：
7.1 年度级别影响因子-在年/月/周/日/小时中可见
    月级别影响因子-在月/周/日/小时中可见
    周级别影响因子-周/日/小时中可见
    日级别影响因子-日/小时中可见
    小时级别影响因子-小时中可见
7.2 根据7.1，你需要重新设计影响因子的影响权重
7.3 你需要找到一个算法让所有值在0-100之间，允许包含小数点后4位
7.4 适当使用差值和抖动算法来让关键数据之间的变化更加平滑和随机
7.5 自定义影响因子本质上就是（叠加函数+生命周期）来进行影响，我举个例子：假设函数是AddScore=(2*healthScore,2.5,202517301212),那么意思就是，从202517301212开始,健康值*2，维持2.5小时.这个时候小时分的变化将会影响小时综合分，也会像蝴蝶效应一样影响后续的所有分直到年为止。
